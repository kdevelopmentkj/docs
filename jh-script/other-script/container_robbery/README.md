# Container\_Robbery

{% embed url="https://kdev-jh.tebex.io/package/container-robbery" %}

{% embed url="https://youtu.be/tcL8U99M4HY" %}

**Container\_Robbery** is a dynamic container heist script that allows players to rob shipping containers scattered across the map. Players use an angle grinder to break into containers, with realistic animations, particle effects, and a comprehensive cooldown system. The script features customizable container zones, multiple container sizes (small, medium, large), various color options, and fully customizable reward systems.

***

**Dependence :**

* None - Standalone system
